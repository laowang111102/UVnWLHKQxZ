# 项目简介 python540

本项目是一款基于Python的招聘推荐系统，集成了爬虫功能与数据可视化，运用协同过滤算法实现智能岗位推荐。

## 技术栈

- **后端框架**：Python + Django
- **推荐算法**：协同过滤（基于用户行为与岗位相似度）
- **数据库**：MySQL
- **数据爬取**：Python 爬虫
- **数据可视化**：Echarts + Django Admin

## 功能模块

### 前台（用户端）

- **岗位信息查询**：支持关键词搜索职位
- **智能推荐**：基于协同过滤算法推荐合适岗位
- **数据可视化**：展示不同岗位的学历要求、工作经验、薪资待遇等

### 后台（管理员端）

- **招聘数据管理**：添加、修改、删除招聘信息
- **爬虫控制**：抓取、更新、管理招聘数据
- **可视化分析**：岗位需求、薪资水平、技能要求等图表展示

## 系统角色

- **用户**：查询岗位信息，接收智能推荐，查看数据可视化分析
- **管理员**：管理招聘数据，控制爬虫更新，进行可视化分析

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Echarts
